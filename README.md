# JSNotepad
Very simple and clean notepad written in JavaScript.

# Usage
Install APK file:
> release/JSNotepad.apk

# Features
 1. **Emojis support**
 2. Colors and size
 3. **Material design interface**
 4. Saving/Opening option
 5. "Alert" function
 6. And more...
